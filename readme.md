
# WASM Servers

WASM is a sandboxed low-level runtime with lots of interest. WASM currently runs in-browser, but is also shipped with NodeJS. Let's create a runtime to allow for mature web services to be built using WASM. Go/C/C++/Rust/Typescript web services can all be compiled to wasm and run with very small memory footprints. 

https://github.com/golang/go/issues/27462
